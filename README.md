# 📖 Quran Verse Generator

A simple, elegant web application that fetches and displays random verses from the Quran in both Arabic (Uthmani) and English (Asad translation).

## 🚀 Live Demo
[View the Live Project Here](https://ruqayaha.github.io/quran-verse/)

## ✨ Features
- **Real-time API:** Fetches data dynamically from the Al Quran Cloud API.
- **Bilingual Display:** Shows the original Arabic text alongside a clear English translation.
- **Copy to Share:** Includes a functional "Copy" button to easily share verses with others.
- **Responsive Design:** Works on both mobile and desktop browsers.

## 🛠️ Built With
- **HTML5 & CSS3:** For the structure and modern "card" UI.
- **JavaScript (ES6+):** Using `fetch` and `async/await` for smooth data retrieval.
- **GitHub Pages:** For hosting and deployment.

## 📝 How to Use
1. Open the live link.
2. Click **"New Verse"** to load a random Ayah.
3. Click **"Copy to Share"** to save the text to your clipboard.
